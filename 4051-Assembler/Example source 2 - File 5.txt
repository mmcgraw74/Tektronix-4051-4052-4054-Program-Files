* EXAMPLE PROGRAM TO PRINT THE PRINTABLE ASCII CHARACTERS
*    (FILE 5)
*
5000          * STARTING ADDRESS
*
ZSTART    LDAA #20 *START WITH ASCII 32 (SPACE CHARACTER)
          STAA 6000
ZCHECK    LDAA 6000
          CMPA #7F *STOP AT ASCII 127 (7F)
          BEQ ZDONE
          JSR F22E *PRINT THE CHARACTER (CTLCHR)
          INC 6000
          BRA ZCHECK
ZDONE     RTS
